**Interfaz web para un sistema de crowdfunding** enfocada en la gestion de
proyectos desde una experiencia mas clara y directa.

**La aplicacion permite** crear, editar y eliminar proyectos de forma sencilla,
con una capa visual pensada para hacer mas accesible la operativa diaria sobre la API.
