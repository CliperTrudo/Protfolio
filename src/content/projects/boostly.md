**Interfaz web para un sistema de crowdfunding** enfocada en la gestión de
proyectos desde una experiencia más clara y directa.

**La aplicación permite** crear, editar y eliminar proyectos de forma sencilla,
con una capa visual pensada para hacer más accesible la operativa diaria sobre la API.
